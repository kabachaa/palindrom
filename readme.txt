There are 3 endpoints created in this application.

http://localhost:8080/
This is the main index page
http://localhost:8080/details
This is to display the results if the word is ture word found in the Mariam Webster Dictionary and is a Palindrome
http://localhost:8080/userWords
This shows all the words that were entered by the user. All the words are stored in a file userWords.txt.

This is Spring Boot application I and can be run using 
"Jave -jar rest-service-0.1.0.war" war file. 

This file is in the Target folder of Palindrome project.

Project can be built locally maven "mvn install" and Java 8 JDK.
